## Task 4 — Sound Design (Web Audio API)

No dependencies needed. Use the Web Audio API directly.

```js
const AudioCtx = typeof AudioContext !== "undefined" ? AudioContext : window.AudioContext;

const playTone = (freq, duration, type = "sine", volume = 0.3) => {
  try {
    const ctx = new AudioCtx();
    const osc = ctx.createOscillator();
    const gain = ctx.createGain();
    osc.connect(gain);
    gain.connect(ctx.destination);
    osc.type = type;
    osc.frequency.value = freq;
    gain.gain.setValueAtTime(volume, ctx.currentTime);
    gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + duration);
    osc.start(ctx.currentTime);
    osc.stop(ctx.currentTime + duration);
  } catch(e) { /* audio not available — silently fail */ }
};

// Sound effects:
const SFX = {
  appOpen:       () => playTone(880, 0.15, "sine", 0.2),
  choiceSelect:  () => playTone(440, 0.08, "square", 0.15),
  damage:        () => { playTone(150, 0.3, "sawtooth", 0.3); },
  heal:          () => { playTone(660, 0.2, "sine", 0.2); setTimeout(()=>playTone(880, 0.2, "sine", 0.2), 100); },
  itemFound:     () => { [523, 659, 784].forEach((f,i) => setTimeout(()=>playTone(f, 0.2, "sine", 0.25), i*80)); },
  goldFound:     () => playTone(698, 0.15, "triangle", 0.2),
  triviaCorrect: () => { [523, 784].forEach((f,i) => setTimeout(()=>playTone(f, 0.15, "sine", 0.25), i*100)); },
  triviaWrong:   () => playTone(220, 0.4, "sawtooth", 0.2),
  victory:       () => { [523, 659, 784, 1047].forEach((f,i) => setTimeout(()=>playTone(f, 0.3, "sine", 0.3), i*120)); },
  death:         () => playTone(110, 0.8, "sawtooth", 0.25),
};
```

Apply sounds:
- `SFX.appOpen()` when opening any app
- `SFX.choiceSelect()` when picking a game choice
- `SFX.damage/heal/itemFound/goldFound` on respective game events
- `SFX.triviaCorrect/triviaWrong` on trivia answers
- `SFX.victory/death` on game endings

Add a mute toggle button in the header:
